Getting started

1
Clone the repo git clone https://github.com/your-org/wiseflow-integration-packages
2
Install dependencies pip install -r requirements.txt
3
Configure credentials cp .env.example packages/01-user-management/.env

Fill in WISEFLOW_BASE_URL, WISEFLOW_CLIENT_ID, WISEFLOW_CLIENT_SECRET.

4
Run a package python packages/01-user-management/wiseflow_users.py
